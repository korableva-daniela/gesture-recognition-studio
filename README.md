Gesture Recognition Studio

Программная система для распознавания статических жестов с использованием свёрточной нейронной сети (CNN)
 в студийном комплексе №1.
Назначение: Автоматизация процесса невербальной коммуникации между исполнителем и звукорежиссёром. 
Система распознаёт 18 типов жестов, отображает результат оператору и ведёт электронный журнал событий.

Функциональные возможности:

- Захват видеопотока с камеры 
- Распознавание жестов с помощью CNN (18 классов)
- Отображение результата в графическом интерфейсе
- Автоматическая регистрация событий в электронном журнале (JSON/CSV)
- Обработка нештатных ситуаций

Технологический стек

- Язык: Python 3.9+
- Фреймворк: PyTorch
- Компьютерное зрение: OpenCV
- Интерфейс: Tkinter
- Контроль версий: Git, GitHub
